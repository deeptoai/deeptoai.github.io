git是功能强大的版本控制系统。可以实现版本回退，多人协作等功能。我写博客只用到以下几个功能：

## git add .
    把工作区的内容添加到版本库的暂存区中
## git commit -m "......"
    把暂存区中的所有修改提交到分支中
## git push origin master
    把分支推送到github
